# Definition of Success (DoS) — MVP Iteration

Проект считается успешным на этапе инициации, если выполнены следующие условия:

### 1. Функциональные критерии
- [ ] Пользователь может добавить задачу, и она сохраняется после перезагрузки страницы.
- [ ] Кнопка "Удалить/Выполнить" работает корректно (элемент исчезает из DOM).
- [ ] Интерфейс не "ломается" при изменении размера окна браузера (базовая адаптивность).

### 2. Процессные критерии
- [ ] Код загружен в ветку `main`.
- [ ] Настроен GitHub Pages, ссылка на сайт открывается.
- [ ] В репозитории есть папки `docs` и `design`.

### 3. Критерии приемки (User Acceptance)
- [ ] Время добавления одной задачи занимает менее 15 секунд.
- [ ] Тестовый пользователь (одногруппник) понял назначение кнопок без текстовой инструкции.
