# Study Planner MVP

## 1. Питч (Pitch)
Студенты часто срывают дедлайны из-за хаоса в учебных чатах и отсутствия единого списка задач. **Study Planner** — это минималистичный веб-инструмент, который позволяет зафиксировать домашнее задание за 10 секунд и держать фокус на приоритетах, снижая тревожность перед сессией.

## 2. Карта стейкхолдеров (Stakeholders)
* **Пользователь:** Студент, которому нужно быстро записать "домашку" и забыть о ней до выполнения.
* **Разработчик (Я):** Отвечаю за реализацию MVP в сжатые сроки (до дедлайна сдачи лабораторной).
* **Внешние факторы:** Преподаватели (задают дедлайны), учебный отдел (меняет расписание).

## 3. Ядро ценности и MVP
**Гипотеза:** Простой список задач без лишних функций (календарей, уведомлений) работает эффективнее сложных планировщиков.
**Сценарий (MVP):**
1. Открыть страницу.
2. Ввести название предмета и дату сдачи.
3. Нажать "Добавить" (задача появляется в списке).
4. Нажать "Готово" (задача исчезает или зачеркивается).

## 4. План проверки гипотез (Hypothesis Verification)
* **Действие:** Дать доступ к прототипу 3 одногруппникам на 2 дня.
* **Метрика:** Пользователь должен успешно создать минимум 3 задачи без моей помощи.
* **Риски и их устранение:**
    * *Риск:* Неудобно вводить дату вручну. *Решение:* Использовать стандартный календарь браузера.
    * *Риск:* Потеря данных при закрытии вкладки. *Решение:* Использовать LocalStorage (память браузера) для MVP.
